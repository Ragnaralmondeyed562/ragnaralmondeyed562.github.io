## 🔍 Troubleshooting
If you encounter issues while using the software, check the following solutions:

*   Application fails to open: Ensure you are running the 64-bit version of Windows. If you use an older computer, try closing background applications to free up system memory.

*   Unexpected transactions: If an agent performs a trade you did not intend, open the logs tab. The logs show the decision-making process for each individual action. Adjust your risk settings if the agent behavior does not match your expectations.
*   Updates: Check the releases page regularly to ensure you have the latest version. We improve the agent swarm capabilities and security protocols with every update.
